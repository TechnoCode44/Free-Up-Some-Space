# Free Up Some Space

Helps you with your storage problems

# Features

Finds the biggest files in a selected folder
- Outputs an Excel file

Deletes permanently deleted files since permanently deleted files can still be recovered
- Uses Multi-Threading

# Install

Clone the repo https://github.com/TechnoCode44/Free-Up-Some-Space

Then on **Windows** run the batch file under <code>scripts/install.bat</code>

Or on **Linux/Mac** run the following in the terminal

<code> pip3 install tk </code>

<code> pip3 install openpyxl </code>

The latest version of Python is recomended but older version from 3.6 and above should be fine (although this is not tested).
